# FantaisiePlayer

プレイヤーの情報を管理するプラグイン<br />
プレイヤーのステータスや設定等をAPIを使って管理します。

:::warning

このプラグインはまだ制作を開始していません。<br/>
仕様が変更される場合がございます。

:::

## ステータスについて

このサーバーにはプレイヤーのステータスがあり、数値によって管理しています。

| ID | 名前（正式名称） | 説明 |
| -- | --- | --- |
| HP | HP（体力） | プレイヤーの体力、0になると死亡します。 |
| MP | MP（魔力） | 主に魔法を使用する際に減ります、MPが足りない場合はHPを使用する場合があります。 |
| STR | 物理（物理攻撃力） | 物理攻撃等に影響します。 |
| INT | 魔法（魔法攻撃力）・知力 | 魔法攻撃等にエ供します、一部役職に影響します。 |
| DEX | 器用（遠距離攻撃力） | 遠距離攻撃等に影響します、職人系の役職の成功率に影響します。 |
| DEF | 防御力 | プレイヤー自身の防御力です、ダメージが受けにくく軽減されます。 |
| AGI | 素早さ（移動速度上昇） | 移動速度が上昇します、一部役職では初期値が低めに設定されます。 |
| CUT | 会心（会心率）・攻撃倍率上昇 | 敵への攻撃力が上昇します、一部役職では初期値が低めに設定されます。 |
| LUK | 幸運 | ドロップするアイテムや量が増加したり、使用の成功率が上がります。例）武器制作の成功率、料理の成功率 |
| LVL | レベル | この値を元にステータスが調整されます。 |
| EXP | 経験値 | この値を元にレベルを調整します。 |

## ステータスのランダム性について

プレイヤーのUUIDから算出した数値で一部のステータスはランダム性が持たれます。<br />
数値の計算方式については非公開といたします。

## 役職について

このサーバーではプレイヤーには役職に就いて<br />
その役職を生かした仕事ができるようになります。

# メイン役職

| ID | 名前 | 説明 |
| -- | --- | --- |
| COOK | 料理人 | 料理を専門にできる役職、料理が得意になる。 |
| MINING_PERSON | 採掘人 | 採掘で鉱石などを見つけやすくなる、採掘速度も通常より高い。 |
| CRAFTSMAN | 職人 | 装飾品や装備、武器などを作れる、職人じゃないとほぼ作れない。 |
| ANGLER | 釣人 | 誰でもなれる、魚を釣って売る事ができる。 |
| FARMER | ファーマー | 素材集め専門、誰でも手に入るが確率が上がったりする。 |
| HUNTER | ハンター | ダンジョンが利用可能、そこでしか手に入らないアイテムもあったり。<br/>移動速度も通常より早くなったり、攻撃力も強くなる。 |
| MERCHANT | 商人 | 外交等で優位に立つことが可能、追加でアイテムが持てる。 |
